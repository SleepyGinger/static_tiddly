2. Lightning-fast linking and navigating

22nd April 2020 at 1:06pm
meta starthere

The basic idea of TiddlyBlink is creating notes and then connecting them by linking.

TiddlyBlink has a number of aids to help you link quickly between notes.

  • When you take notes, typing [[ triggers an autocomplete. If the tiddler exists, use the down arrow to navigate to it, and hit enter to create the link. If the tiddler does not exist, type the ]] to finish creating the link manually.
  • There is also a linkify button in the editor toolbar ([[*]]). Use the button to turn selected text into a link.
  • There is also a link re-naming plugin, so if you change the title of a tiddler, this will automtically change any links to that tiddler found in other tiddlers.
  • Tiddlers have new-tiddlers-from here buttons ( and ) that create new tiddlers linking to the present tiddler. This saves you a step and allows you to focus on writing. There are also similar buttons in the sidebar for loose notes. The tiddlers created with the (clock icon) buttons have date and time stamps as titles, so you don't even need to think about writing titles.
  • You can choose from one of four ways to view references: view them as simple links, view the entire tiddler (simple transclusion), view links that each open to reveal the entire tiddler (collapsible transclusion), or view the relevant parts of the tiddlers as highlighted search results in context.
  • Known issues: References will not be visible in tiddlers with titles with both full parentheses [[The Odyssey (Homer)]] or titles with initial letters that have accents. [[Ã‰xodo]]

4. Topics, sources and hiding?

22nd April 2020 at 1:06pm
meta starthere

I have created some optional features for TiddlyBlink. Feel free to ignore them and just create notes and connect them by links. But you may find some of these features helpful:

  • I tag tiddlers about my sources (tiddlers for book chapters, dictionary articles, webpages, etc) with the tag sources. That way they show up in the sources tab in TiddlyBlink's home page, and I can go directly to a list of them there.
  • Hiding: I like to create long outlines and table of contents tiddlers. But in the simple transclusion view of the references, these are an eyesore. So I created a tag hide to keep them from appearing in the reference section below the tiddlers. You can use this tag to hide any tiddler you don't want to appear there.
    • You can also use the tag outlines for the same effect (left this in for backwards compatibility).
  • Notes are tiddlers created with the clock icon buttons. They have a date and time stamp as a title. They are for when you want to create a note but don't want to waste time thinking up a title.
  • Topics are all the other tiddlers. To be precise, topics are any tiddler not tagged sources or outlines, and that don't have 202 in the tiddler title.
  • Meta tiddlers are the tiddlers for documentation of TiddlyBlink. They are tagged meta.

5. My own notetaking process in TiddlyBlink

22nd April 2020 at 1:06pm
meta starthere

I am using TiddlyBlink for taking notes on my reading. I offer the following as one way TiddlyBlink can be used:

  • I start by creating a tiddler for the book chapter, encyclopedia or journal article, or webpage that I am going to read.
    • I click the + button in the sidebar to create the tiddler. I tag it sources and give it an appropriate name, and save the tiddler.
    • I find that using chapters rather than books is better both for motivation and for precise organization.
  • I then take notes. I could take notes in the tiddler itself, but I prefer to take short notes using the clock icon at the top of the tiddler.
    • Before or after the link in the new note, I type or paste content. I use links to indicate related topics, and indicate the page number from the source, if applicable.
    • I save the note and create tiddlers from the missing links to the related topics.
    • Last step, optional, but I find it helpful: I go to the 'Recent' tab in the sidebar, and copy the new topic links, and create outline tiddlers (tagged outlines or hide). I paste the links and manually rearrange them as links in an outline. The tag prevents the outline from appearing below the tiddlers linked in the outline.
      • Another way to do this would be to tag each new topic tiddler with the categories it belongs to. But remember that a large number of tags will eventually slow your file down.

Possible reorganizing steps:

  • Renaming note tiddlers in order to stack them in an order of your choosing

Finished product:

  • The sources should appear in the sources tab in the home page, the topics should appear in the topics tab, and any outlines in the outlines tab. No datetimestap notes should appear in any of the tabs. And outline tiddlers should not appear in the links or transclusions under the tiddlers.
  • Easy moving from a topic tiddler to the note itself or more likely the source from which the note was taken.

TiddlyBlink

22nd April 2020 at 1:06pm
.sidebyside meta tb

Welcome to TiddlyBlink! TiddlyBlink is an adaptation of TiddlyWiki with the goal of helping you see connections between your ideas, and move quickly from one idea to another. It was inspired by the bi-directional linking found in Roam (https://roamresearch.com/), but built with capabilities already available in TiddlyWiki (https://tiddlywiki.com). See my example file here.

The basic idea:

Below each tiddler (note card) is a list of links to any other tiddlers that link to the current tiddler's title in their text field. There are four options for viewing those links, as seen in menu above or in the TiddlyBlink menu in the sidebar. So as you create tiddlers for sources, notes and topics, you can link them all to each other and navigate between them in the 'blink' of an eye!
